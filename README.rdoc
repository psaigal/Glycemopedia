== Glycemopedia

Glycemopedia is a web application that enables users to make intelligent food choices based on glycemic index of foods consumed, which can help in the prevention of diabetes and help blood sugar regulation. The skills and technologies used in this project are:

* Ruby on Rails

* TSV Parsing

* AJAX

* Bootstrap

* Rails-jquery-autocomplete

To view this app, please visit the following link: https://glycemopedia.herokuapp.com/.
